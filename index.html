Projects

EdmReads

Edmonton Reads is a free website that facilitates reading for the citizens of Edmonton. Featured by the City of Edmonton, this website showcases the top 10 most popular book each week. The backend was developed using Django, Python, and Google APIs while the frontend was developed using Bootstrap, HTML, and CSS along with Javascript to query reading lists.

YegEvents

YegEvents is an iOS app that showcases nearby events happening in the City of Edmonton. Featured by Harvard University's CS50, this pre-production app shows local events on a map and gives you a description and directions with just a tap. This app was developed in Xcode using UIKit, MapKit, and the City of Edmonton API to access events. This app introduced me to developing in Swift and parsing JSON API requests in an appealing format.
